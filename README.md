# 🎲 Scrabble (Java Edition)

## 🧩 Overview
This project is a Java-based implementation of the classic board game **Scrabble**. It recreates the core gameplay experience, including tile placement, word validation, scoring, and turn-based play. Created for Mr. Yee's AP CS2 Class in 2024.

---

## ✨ Features
- 🟫 Full Scrabble board implementation  
- 👥 Turn-based multiplayer gameplay  
- 🌐 Word validation system using an online API  
- 📊 Score tracking  
- 🔤 Tile distribution and rack management  

---

## 📁 Project Structure
```text
Scrabble/
│
├── 📁 src/                     
│   ├── Board.java             # Game board logic
│   ├── BonusTile.java         # Bonus tile behavior
│   ├── Player.java            # Player model
│   ├── Scrabble.java          # Main game controller
│   ├── SpecialTiles.java      # Special tile definitions
│   ├── Tile.java              # Tile representation
│   ├── TileBag.java           # Tile distribution system
│   └── Window.java            # GUI / window handling

├── 📁 assets/

│   ├── Player1Wins.jpg
│   ├── Player2Wins.jpg
│   └── TIE.jpg
│
```

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/michaellikens/scrabble
   cd scrabble
   ```

2. Compile the project:
   ```bash
   javac *.java
   ```

3. Run the game:
   ```bash
   java Main
   ```
